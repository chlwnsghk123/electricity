## 자동 검출

`app.js`는 `q.has_image === true` 인 문제에 대해 `images/<examId>/<no>.png` 경로를 `<img>` 태그로 삽입합니다. 파일이 없으면 onerror로 숨김 처리되고 기존 `image_note` 텍스트가 대신 표시됩니다.

## AI 질문 시 이미지 첨부

AI에게 질문할 때 해당 문제의 이미지가 있으면 자동으로 함께 전송됩니다(Gemini 멀티모달). 이미지 파일을 넣기만 하면 별도 설정 없이 동작합니다.
