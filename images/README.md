# 이미지 리소스

문제에 포함되는 그림/도식을 저장하는 폴더.

## 규칙
- 회차별 폴더: `images/<examId>/` (예: `images/electricity-2022-r2/`)
- 파일명: `<문제번호>.png`
- 포맷: PNG 권장

## 회차별 필요 목록
각 회차 폴더 안의 `README.md` 참고.

| 회차 | 상태 |
|------|------|
| electricity-2022-r2 | 목록 작성 완료 (12장 필요) |
| 그 외 | 회차 데이터 추가 시 자동 생성 예정 |

## 작동 방식
- `q.has_image === true` 인 문제는 `images/<examId>/<no>.png`를 `<img>`로 표시합니다.
- 파일이 없으면 자동으로 숨기고 `image_note` 텍스트만 노출합니다.
- AI 질문 전송 시, 해당 문제 이미지가 있으면 Gemini에 함께 전송합니다(멀티모달).
