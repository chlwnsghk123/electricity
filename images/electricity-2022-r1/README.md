# 전기기사 2022년 1회 — 필요 이미지 목록

이 폴더에 아래 번호로 **`<번호>.png`** 파일을 넣어주세요. (예: `4.png`)

파일 경로 규칙: `images/electricity-2022-r1/<번호>.png`

지원 포맷: PNG 권장.

## 아직 미정

회차 데이터가 아직 등록되지 않았습니다. 텍스트 소스가 입력되면 `has_image: true` 인 문제 번호 목록이 이 문서에 자동으로 갱신될 예정입니다.

## 자동 검출

`app.js`는 `q.has_image === true` 인 문제에 대해 `images/<examId>/<no>.png` 경로를 `<img>` 태그로 삽입합니다. 파일이 없으면 onerror로 숨김 처리되고 `image_note` 텍스트가 대신 표시됩니다.

## AI 질문 시 이미지 첨부

AI에게 질문할 때 해당 문제의 이미지가 있으면 자동으로 함께 전송됩니다(Gemini 멀티모달).
