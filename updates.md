# Updates

## v1.05 — 2026-04-21
- **홈 구조 개편**: 회차 선택 전용 화면. 2022년 2회만 활성, 나머지는 "준비 중" 라벨로 비활성. 버튼 느낌을 줄여 리스트 아이템 스타일로 전환.
- **모드 선택 화면 분리**: 회차 선택 후 진입. 모드 3개(학습 / 랜덤 학습 / 모의고사). 모든 모드 카드 동일 스타일(primary 강조 제거).
- **모의고사 개편**: 리스트를 건너뛰고 바로 상세 진입. 과목별 20문제(정식) 또는 10문제(하프) 랜덤 추출. 순서도 셔플. 마지막 문제 "시험 제출하기" 버튼. 시간: 정식 150분 / 하프 75분.
- **랜덤 학습 추가**: 과목 선택 → 해당 과목 문제를 무작위 1문제씩 연속 제공. 한 바퀴 돌면 풀 리셋.
- **과목별 모드 제거, 홈의 북마크 단축 진입 메뉴 제거**.
- **설정 시트**: 헤더 우측 ⚙ 버튼 → 설정 바텀시트. "진행도 초기화"는 확인 모달 경유. 기존 하단 초기화 버튼 삭제.
- **6색 태그 + 의미 라벨**: gray(미착수/지우기), green(맞춤), red(틀림), orange(학습 필요), yellow(다시 풀기), blue(중요). 상세 뷰 피커와 리스트 필터 칩에 의미 라벨 노출.
- **자동 태그**: 학습/랜덤 모드에서 정답→green, 오답→red 자동 부여. 모의고사는 제출 시점에 일괄 부여.
- **답 저장 정책 변경**: localStorage에 답을 더 이상 저장하지 않음. 학습/랜덤 모드의 답은 상세 뷰를 벗어나면 소멸. 모의고사 답은 제출까지 메모리에서 유지. (LS 스키마: `cbt_tags_v1` → `cbt_tags_v2`, `cbt_answers_v1` 제거.)
- **메모 기능**: AI 답변 옆 "📝 메모에 저장" 버튼 → 해당 문제의 메모 목록에 저장. 상세 뷰 하단에 메모 목록 + 삭제 버튼. 스키마: `cbt_notes_v1`.
- **KaTeX 수식 렌더**: 문제 본문·선택지·정답 라벨·AI 답변·메모 모두 `$...$`/`$$...$$` 자동 렌더. AI 시스템 프롬프트에도 LaTeX 권장 지시 추가.
- **이미지 폴더 구조**: `images/<examId>/<no>.png`. `has_image: true` 문제는 자동으로 `<img>` 삽입(없으면 숨김). 회차별 필요 이미지 목록을 `images/<examId>/README.md`에 기재.
- **AI 멀티모달**: 문제가 `has_image` 이고 이미지 파일이 존재하면 base64로 읽어 Gemini에 함께 전송.
- **앱 아이콘 + PWA**: `favicon.svg`, `icon.svg`, `manifest.webmanifest` 추가 (블루 배경 + 번개). 간단형.
- **상세 뷰 헤더에 타이머**: 모의고사 모드에서만 ⏱ 표시, 10분 이하 시 경고색.
- **제목 스타일 정리**: 회차 항목을 "버튼"이 아닌 "리스트" 느낌으로 조정.
