- **대화 영속화**: 문제별로 `cbt_ai_v1` 에 저장. 시트를 닫아도 대화 유지, 다음에 같은 문제에서 열면 그대로 복구. 응답 도착 즉시 자동 저장.

## v1.06 — 2026-04-21
- 2022년 2회 데이터 전면 LaTeX 변환: 유니코드 수학 기호(∂, μ, Ω, π, ε, λ, σ, δ, ∇, ⋅, ², ³, ⁻¹, ₀ 등)를 KaTeX용 `$...$` LaTeX 구문으로 치환. 질문 본문 42개 / 선택지 86개 변경.
- 변환 처리: 그리스 문자, 상·하첨자 시퀀스, 루트(`√(...)`→`\sqrt{...}`), 결합 모자(`x̂`→`\hat{x}`), 논리 바(`Ā`→`\bar{A}`), 스크립트 글자(`ℒ`→`\mathcal{L}`), 상첨자 문자(`ᵗ`,`ⁿ` 등). 그리스 문자 + 단일 소문자 경계는 물리 관례상 subscript(`μr`→`\mu_r`)로, 그 외는 공백(`μS`→`\mu S`)으로 분리.
- 한글 사이의 middot(`정·역전`)은 `\cdot`으로 바뀌지 않도록 전처리 유지.
- 한글 혼합 선택지는 토큰별로 수식 부분만 감싸고 본문 텍스트는 그대로 보존.
- 변환 스크립트: `tools/latexify.cjs` (일회성, 재실행 시 동일 결과).
